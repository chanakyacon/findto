Searchy

A simple tool that makes it more productive search on web

A new experience to help search on web

Searchy is a simple tool that makes it easier and more productive to search on web. Allow you to use the power of multiple search engines to quickly search on web and modules to search any content directly on your favorite source.

Functions

All you can do with Searchy.

Multiple Search Engines

Google, Bing, Yahoo, DuckDuckGo, WolframAlpha, Baidu and Yandex.

Voice Search

Allows to use voice recognition to search in many sites.

Search by Modules

Web, Social, Video, Music, Images, News, Shopping, Local and Torrents.

Dynamic Info

Displays dynamic information related to time and weather.

Auto Suggest

Predictions of possible terms you can use to search, related to what you’re typing.

Accessibility

Everyone should be able to access and enjoy the functions of this web search tool.

Total Privacy

Zero bytes of user data are collected, like search history, location and cookies.

No Ads and Tracking

No annoying advertising is displayed and not tracking user behavior.

Open Source

You are free to contribute and help improve this search tool.
